"Словарь терминов, используемых в тестировании программного обеспечения"

"Слова на букву A"

  • Accept – принимать.
  • Account – аккаунт/учетная запись.
  • Account preferences – настройки учетной записи.
  • Actual result – фактический результат.
  • Additional information – дополнительная информация.
  • Align – выровнять.
  • Appear – появляться.
  • Application crash - экстренное завершение работы приложения.
  • Assign - назначать.
  • Attached files – прикрепленные файлы.
  • Аuthorization – авторизация.
  • Authentication – аутентификация.

"Слова на букву B"

  • Banner – баннер.
  • Bug report – баг-репорт/дефект/отчет об ошибке.

"Слова на букву C"

  • Checkbox – чекбокс (флаговая кнопка).
  • Click the button/link – нажать кнопкой мыши на кнопку/ссылку.
  • Close the window – закрыть окно.
  • Create an account – создать аккаунт.
  • Combo box/drop-down list – раскрывающийся список.
  • Сurrent/new password – текущий/новый пароль.

"Слова на букву D"

  1. Data – данные.
  2. Default cursor – курсор по умолчанию.
  3. Delete – удалять.
  4. Device – устройство.
  5. Description – описание.
  6. Display – отображается.
  7. Drop-down menu – выпадающее меню.

"Слова на букву E"

  1. Edit field – поле редактирования.
  2. E-mail – электронный адрес.
  3. Enter – ввести.
  4. Error message – сообщение об ошибке.
  5. Expected result – ожидаемый результат.
